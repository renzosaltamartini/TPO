# SkyBridge Airlines - Sistema de Gestión de Vuelos

## Alcance del proyecto

El proyecto consiste en un sistema desarrollado en Python para gestionar vuelos de la empresa ficticia SkyBridge Airlines.

El sistema permite registrar, eliminar, modificar y visualizar vuelos mediante un menú interactivo en consola. La información se almacena en memoria utilizando una matriz, representada como una lista de listas.

Para cada vuelo se administra la siguiente información:

- Código de vuelo.
- Destino.
- Horario de salida.
- Cantidad de pasajeros.
- Peso total del equipaje.
- Estado operativo.
- Tipo de vuelo.

Las funcionalidades principales del sistema son:

- Alta de vuelos.
- Baja de vuelos cancelados.
- Modificación de vuelos existentes.
- Informe general de vuelos ordenado por cantidad de pasajeros y, en caso de igualdad, por destino.
